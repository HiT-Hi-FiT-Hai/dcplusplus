Downloads

This list shows you the status of your current downloads (from other peers to you).

Columns

Filename
The name of the file beeing downloaded
Path
Shows where the current download will be saved when finished
Status
Shows the current status of the download which can be downloading from one or more user or wating for a free slot
Time Left
Displays the current estimated time left
Speed
The actual download speed
Done
Total number of downloaded bytes so far
Size
Size of the file beeing downloaded
