Colors and Sounds

Note: Most of these options require that you restart DC++. (Fonts, colors and icons are loaded at startup only. Window settings are loaded when opening a new window.)

Colors

Set Window Color
This setting determines the background color of the windows in DC++.
Set Text Style
This setting determines the color and font used in many DC++ windows.
Uploads
This will set the coloring for the upload progress bars, if you have them enabled in Advanced. (default: RGB(205, 60, 55))
Download
This will set the coloring for the download progress bars, if you have them enabled in Advanced. (default: RGB(55, 170, 85))

Sounds

Make an annoying sound every time a private message is received
When set, the system's "Default Beep" sound will play when a private message is received.
Make an annoying sound when a private message window is opened
When set, the system's "Default Beep" sound will play when a private message window is opened.

Bold options

Set Finished Download Tab Bold When an Entry is Added
When a download completes, the Finished Downloads tab will turn bold if this option is selected.
Set Finished Upload Tab Bold When an Entry is Added
When a upload completes, the Finished Uploads tab will turn bold if this option is selected.
Set Download Queue Tab Bold When Contents Change
This option controls whether or not the Download Queue's tab will highlight itself when a queue item changes state or finishes.
Set Hub Tab Bold When Contents Change
When a chat message appears in a hub that's not focused the tab will become bold if you have this option selected.
Set PM Tab Bold When Contents Change
When a private message appears that's not focused the tab will become bold if you have this option selected.
Set search Tab Bold When Contents Change
If more searchresult appear in an open searchwindow that's not focused the tab will become bold if you have this option selected.
