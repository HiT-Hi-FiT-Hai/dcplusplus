Finished uploads

Note that file list logging must be enabled for them to appear in this window.

Columns

Time
The date and time the file finished uploading.
File name
The file name of the uploaded file.
Path
The path to the uploaded file.
Nick
To whom the file was uploaded.
Hub
Which hub the user was on.
Size
The size of the file uploaded.
Speed
The avarage speed for the file.
CRC Checked
Whether or not the file was CRC checked. Note that Enable automatic SFV checking must be enabled, and the file need to have an accomponaying SFV file..

Status bar

# Items
The amount of items that you have uploaded (sent).
# bytes
The amount in bytes that you have uploaded (sent).
# bytes/seconds
The average speed of the files you have uploaded (sent).
