Appearance

Note: Most of these options require that you restart DC++. (Fonts, colors and icons are loaded at startup only. Window settings are loaded when opening a new window.)

Colors

Set Window Color
This setting determines the background color of the windows in DC++.
Set Text Style
This setting determines the color and font used in many DC++ windows.
Uploads
This will set the coloring for the upload progress bars, if you have them enabled in Advanced. (default: RGB(205, 60, 55))
Download
This will set the coloring for the download progress bars, if you have them enabled in Advanced. (default: RGB(55, 170, 85))

Options

Filter kick and NMDC Debug Messages
When enabled, kick and debug chat messages (sent by other clients) are hidden from main chat. These messages are generally not useful and make reading the chat more troublesome.
Minimize to Tray
When enabled, minimizing DC++ will cause it to appear as an icon in the windows system tray.  If you left click, it will be returned to its previous status.  If you right click on the icon, it will give you a small menu.
Show Timestamps in Chat By Default
This option will show timestamps on chat lined in newly opened hubs.  In order to show them in an already open hub, use the /ts chat command. To customize the format of the timestamps, see the Set Timestamps option below.
Confirm Application Exit
Enabling this option will cause DC++ to prompt you before exiting.
View Status Messages in Main Chat
Show some messages destined for the Status Bar in main chat as well. It's generally a good idea to leave this enabled.
Show Joins / Parts in Chat By Default
Show an entry in main chat whenever someone joins or leaves the hub. Use the /showjoins chat command to enable or disable it on already open hubs. On large hubs, you don't want this enabled.
Only Show Joins / Parts for Favorite Users
With this option enabled, only the joining and parting of Favorite Users are shown.
Use System Icons When Browsing Files
If you have this option on, DC++ will use the per-extension icons from Windows Explorer.  Turning on this option slows browsing file lists a bit.
Use OEM Monospaced Font for Viewing Text Files
Enable this option to use a monospaced font more suitable for viewing NFO files in the built-in text viewer (the "View as Text" function).
Set Finished Manager(s) Tab Bold When an Entry is Added
When a download or upload completes, the Finished Downloads/Uploads tab will turn bold if this option is selected.
Set Download Queue Tab Bold When Contents Change
This option controls whether or not the Download Queue's tab will highlight itself when a queue item changes state or finishes.
Set Hub/PM Tab Bold When Contents Change
When a private message or chat message appears in a hub that's not focused, the tab will become bold if you have this option selected.
Get User Country
When selected, country codes for your uploaders and downloaders appear in the Transfer View.  The GeoIPCountryWhois.csv database is based on one provided by MaxMind, but has been reformatted so the updated databases they provide are not compatible with DC++.

Miscellaneous

Default Away Message
This is the default away message that will appear when someone PMs you for the first time while you are in the away mode.  You can activate away mode by using the /away <message> and /back chat commands or Auto-away on Minimize option.
Set Timestamps
This lets you select the format of the time stamps that can appear in the chat and private message windows.  For the available formatting variables, please see Logs and Sounds.
Language File
DC++ will read this XML translation file and load the strings in it, allowing you to use DC++ in whatever language the translators have made available.  Links to many translations can be found on the Help Menu.
