Finished downloads

Note that file list logging must be enabled for them to appear in this window.

Columns

Time
The date and time the file finished downloading.
File name
The file name of the downloaded file.
Path
The path to the downloaded file.
Nick
The source's nick name.
Hub
Which hub the source was on.
Size
The size of the file downloaded.
Speed
The avarage speed for the file.
CRC Checked
Whether or not the file was CRC checked. Note that Enable automatic SFV checking must be enabled, and the file need to have an accomponaying SFV file..

Status bar

# Items
The amount of items in that have finished.
# bytes
The amount in bytes that have finished.
# bytes/seconds
The avarage speed of the total finished files.
