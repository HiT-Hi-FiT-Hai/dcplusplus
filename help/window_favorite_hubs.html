Favorite hubs

Columns

Auto connect / name
Specifies what name the hub has. The checkbox indicates if DC++ should auto connect to the hub on startup. If the box is checked, DC++ will automatically connect to the hub on startup.
Description
A brief description of the hub.
Nick
If you would like to use a special nick for a specific hub. Leave this blank for default.
Password
If you have a password attached together with your nick. Used for operators och other registered people.
Server
The IP or DNS for the hub.
User Description
Brief description of the user account.

Buttons

New
Create a new favorite hub.
Properties
Change pre-existing hub properties.
Remove
Removes the chosen favorite hub.
Move up
Move up a hub in the list. If several hubs have auto connect enabled, the highest hub will be connected to first.
Move down
Move down a hub in the list. If several hubs have auto connect enabled, the lowest hub will be connected to last.
Connect
Connects to chosen hubs.
