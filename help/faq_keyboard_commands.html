Keyboard shortcuts

This page enables you to see which keyboard shortcuts exist in DC++.

F1
Opens the help file.
Alt
Marks the menu so you can you the arrows to navigate.
Makes the lower status bar invisible. The bar will be visible again once you press Alt again or you open a new window. Note that this is a Windows feature.
Tab
Completes nicks in chat.
Change positioning view where there is split view. You can find the split view in the file list and in the download queue.
Jump between input boxes.
Shift
Pressing shift while startup will disable auto connection to favorite hubs.
Ctrl + 1
Enable or disable the toolbar.
Ctrl + 2
Enable or disable the status bar.
Ctrl + 3
Enable or disable the transfer view.
Ctrl + C
Copy text. Note that this is a Windows feature.
Ctrl + D
Open download queue
Ctrl + E
Refresh file list.
Ctrl + F
Open favorite hubs.
Ctrl + N
Open notepad.
Ctrl + L
Open file list.
Ctrl + P
Open Public hubs.
Ctrl + Q
Quick connect.
Ctrl + R
Reconnect.
Ctrl + S
Open search.
Ctrl + T
Follow last redirect.
Ctrl + U
Open favorite users.
Ctrl + V
Paste text. Note that this is a Windows feature.
Ctrl + X
Cut out text. Note that this is a Windows feature.
Ctrl + Y
Redo text edit. Note that this is a Windows feature.
Ctrl + Z
Undo text edit. Note that this is a Windows feature.
Ctrl and + (on the numeric keyboard)
If certain columns, e.g. those in the user list, search and the transfer view are not visible, pressing Ctrl and + (on the numeric keyboard) will resize the columns to the width necessary to accommodate the widest text within.
Ctrl + Tab
Move between tabs. Note that this is a Windows feature.
Ctrl + up
Move up in chat messages. (only in main chat)
Ctrl + down
Move down in chat messages. (only in main chat)
Alt + C
In public hubs: connect to a hub.
In favorite hubs: connect to a favorite hub.
Alt + D
In ADL Search: move down an item.
In favorite hubs: move down a favorite hub.
Alt + I (only public hubs)
Set focus to the filter.
Alt + N
In ADL Search: add a new item.
In favorite hubs: add a new favorite hub.
Alt + P
In ADL Search: edit an item.
In favorite hubs: edit a favorite hub.
Alt + R
In Public hubs: refresh the list.
In ADL Search: remove an item.
In favorite hubs: remove a hub.
Alt + S
Send chat message.
Alt + T (only ADL Search)
Open the FAQ about the ADL Search.
Alt + U
In ADL Search: move up an item.
In favorite hubs: move up a favorite hub.
+ (on the numeric keyboard)
In the download queue, pressing + (on the numeric keyboard) sets a higher priority on a download.
- (on the numeric keyboard)
In the download queue, pressing - (on the numeric keyboard) sets a lower priority on a download.
